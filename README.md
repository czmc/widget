#插件小集合
展示：

<img src="/preview/1.jpg" width = "324" height = "576" alt="图片1" align=center />

主要集成一些UI组件便于依赖使用：
v1.0.2
------
    CircleImageView
    圆形图片View。

------
    EditTextWithCounter
    带有统计限制字数和统计字数的EditText。

------
    GridViewWithHeaderAndFooter
    扩展原生GridView，提供类似于ListView的顶部视图和底部视图的功能。

--------
    InnerGridView
    解决ScrollView或ListView嵌套GridView嵌套只显示一行的问题。

--------
    SwitchView
    酷炫的切换开关
 v1.0.3  
--------
    ViewPagerIndicator
    viewpager索引指针
    
--------
    FloattingButton
    悬浮在window上的按钮，可跟随手指移动
    
    
##依赖使用
    compile 'me.czmc.library:widget:1.0.3'
